# 2027학년도 논술전형 한눈에 보기 (상담용)

> 2027학년도 대학별 논술 전형 상세 정보를 한눈에 비교할 수 있는 입시 컨설팅 도구입니다.

**제작 : 숭신고등학교 진로전담교사 김강석**

---

## 주요 기능

- 최대 6개 대학/계열 목표 설정
- 2개 대학 실시간 비교 분석
- 논술 비율, 수능최저, 시험일시, 논술 유형·출제범위, 등급별 환산점수 확인
- 리포트 PDF 다운로드 (6개 대학 요약본)

---

## 로컬 실행 방법

**사전 요구사항:** Node.js 18 이상

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

```bash
# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

---

## 기술 스택

- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- Framer Motion
- jsPDF + html-to-image

---

## 데이터 출처

`src/data.ts` 파일에 수록된 데이터는 각 대학의 2027학년도 논술전형 모집요강을 바탕으로 작성되었습니다.  
정확한 내용은 반드시 **해당 대학 홈페이지의 최종 모집요강**을 확인하시기 바랍니다.
